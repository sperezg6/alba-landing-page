# Key Messaging Framework

## The Shift:

| OLD FRAMING (Avoid) | NEW FRAMING (Use) |
|---------------------|-------------------|
| "We redesigned your website" | "Your patients can now..." |
| "Before vs After" | "What if they could..." |
| "New features" | "Solutions to real needs" |
| "Modern design" | "An experience that cares" |
| "We built a chatbot" | "Answers at 2am" |
| "Online booking system" | "3 clicks, any hour" |

## Core Value Props:

1. **Chatbot** → "Porque tus pacientes necesitan más" (respuestas 24/7)
2. **Booking** → "Porque el tiempo de tus pacientes importa" (sin fricción)
3. **Design** → "Porque la primera impresión importa" (abraza, no solo informa)
4. **Mobile** → "Porque están en su celular" (donde viven)

## Emotional Through-Line:

"Más que una clínica, somos familia"
→ Your website should feel like family too
→ Warm, caring, always there
→ "Tu nuevo amanecer digital"
